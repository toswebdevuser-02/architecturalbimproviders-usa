# **App Name**: BIM Architecture Insights

## Core Features:

- Content Display: Display blog content in a readable, one-page layout.
- Prominent Heading: Provide a prominent heading showcasing the blog title: Top 5 Architectural BIM Service Providers Reshaping American Design Requirements.
- Smooth Scrolling: Implement smooth scrolling for easy navigation between sections.
- External Links: Make all external links to service providers open in new tabs for user convenience.
- AI Content Suggestion: Use an AI tool to suggest possible content updates, additional topics and rewrite headlines.
- Responsive Layout: Implement responsive design to adapt to different screen sizes.

## Style Guidelines:

- Primary color: Soft blue (#72A0C1) to evoke trust and professionalism in architecture.
- Background color: Light gray (#F0F2F5), offering a clean, neutral backdrop that ensures readability and visual comfort.
- Accent color: Warm orange (#D98828), used sparingly to highlight key interactive elements.
- Headline font: 'Playfair', a modern serif font with an elegant feel.
- Body font: 'PT Sans', a modern humanist sans-serif that complements 'Playfair' for body text.
- Minimalist icons to represent each service provider's key offerings or specializations.
- Clear sections for each of the Top 5 BIM service providers.
